# Air Airborne Agency Website

A high-performance, cinematic smooth-scroll web application for Airborne Agency.

## 🚀 Live Local Preview
To preview locally:
- Run `powershell -ExecutionPolicy Bypass -File .\server.ps1`
- Open `http://localhost:3000` in your browser.

---

## 🌐 Deploying to Vercel

1. **Import Repository**:
   - Go to [Vercel Dashboard](https://vercel.com/dashboard) and click **Add New Project**.
   - Select the GitHub repository `danir1252008-bit/Air-Borne-Website`.

2. **Deploy**:
   - Framework Preset: **Other** / **Static Site**
   - Root Directory: `./`
   - Click **Deploy**. Vercel will automatically host the static files with clean URLs using the included `vercel.json`.

---

## 🏢 Deploying to Hostinger (or any HTML Web Host)

1. Log into your **Hostinger hPanel**.
2. Navigate to **File Manager** -> `public_html`.
3. Upload all project files (`index.html`, `pricing.html`, `portfolio.html`, `styles.css`, `app.js`, logo, and image frames).
4. Your website will be instantly live on your domain!

---

## 🛠 Features
- 222-frame cinematic canvas scroll animation
- Dynamic pricing estimator with `#7a5725` active service toggles
- Instant WhatsApp direct booking integration
- Fully responsive across desktop, tablet, and mobile devices
